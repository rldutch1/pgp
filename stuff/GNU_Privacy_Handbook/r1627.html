charset

Name

charset — set the name of the native character set.
charset name

Description

This is used to convert some strings to UTF-8 encoding. Valid values for name are

NameCharacter set
iso-8859-1The default Latin 1 set
iso-8859-2The Latin 2 set
koi8-rThe usual Russian set (rfc1489)
