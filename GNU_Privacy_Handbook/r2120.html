not-dash-escaped

Name

not-dash-escaped — changes the format of cleartext signatures
not-dash-escaped

Description

This option is useful for cleartext signatures on patch files. Messages signed this way should not be sent via email because all spaces and line endings are hashed too. This option cannot be used for data which which has five dashes at the beginning of a line. A special header line is used to tell GnuPG that this option has been used.
